A cosa serve bootstrap?

Bootstrap 5 is the newest version of Bootstrap, which is the most popular HTML, CSS, and JavaScript framework for creating responsive, mobile-first websites.

Cosa devi importare per creare una pagina web con bootstrap

< link href= "https://cdn.jsdelivr.net/npm/bootstrap@5.3.1/dist/css/bootstrap.min.css" rel="stylesheet" >

Cos’è un CDN? Che vantaggi comporta rispetto a scaricare bootstrap sul tuo sito?

Many users already have downloaded Bootstrap 5 from jsDelivr when visiting another site. As a result, it will be loaded from cache when they visit your site, which leads to faster loading time. Also, most CDN's will make sure that once a user requests a file from it, it will be served from the server closest to them, which also leads to faster loading time.

A cosa serve questo attributo del tag meta? < content="width=device-width", initial-scale=1 >

La parte width=device-with imposta la larghezza della pagina in modo che segua la larghezza dello schermo del dispositivo (che varierà a seconda del dispositivo). La parte partial-scale=1 imposta il livello di zoom iniziale quando la pagina viene caricata per la prima volta dal browser.

Che differenza c’è tra Container e Container Fluid (prova il container fluid creando una pagina chiamata BTS_1.1.1.html)

la differenza tra container e container fluid e che container usa solo una parte della pagina, invece quello fluid prende tutta la larghezza che puo usara.
